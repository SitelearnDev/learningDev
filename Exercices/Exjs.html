Instructions de base :

Sachant que les allocations familiales d’un employé sont attribuées comme suit :

1. 200 dh pour chacun des trois premiers enfants.

2. 100 dh pour le quatrième et le cinquième.

3. 0 dh pour les autres Ecrire un script Javascript qui permet de calculer puis d’afficher les allocations familiales d’un employé selon le nombre d’enfants entré par l’utilisateur

Exercice 2 :

L'itération :
Écrivez les fonctions suivantes :
1. factorial qui prend un nombre en entrée et retourne sa factorielle.
2. isAnagram qui vérifie si deux chaînes sont des anagrammes l'une de l'autre.
3. checkPalindrome qui vérifie si une chaîne de caractères est un palindrome.
4. generateFibonacci qui génère les n premiers termes de la séquence de Fibonacci. Sachant que la suite de Fibonacci est donnée comme suit :
Si F(n) représente le nième terme de la suite de Fibonacci, alors:
• F(0)=0
• F(1)=1
• F(n)=F(n−1)+F(n−2) pour n>1

Exercice 3 :

Tableaux :
Écrivez les fonctions suivantes :
1. sortArray qui trie un tableau d'entiers de manière ascendante.
2. findMissingNumber qui trouve le nombre manquant dans une séquence d'entiers consécutifs passée en paramètre.
3. rotateArray qui effectue une rotation à gauche sur un tableau donné d'un nombre spécifié de positions. Exemple d’exécution : rotateArray([1,2,3,4,5,6,7,8,9],3) => [4,5,6,7,8,9, 1,2,3]
4. avgArray qui renvoie la moyenne des valeurs du tableau passé en paramètre.
5. countUpToAvg qui revoie le nombre des valeurs supérieurs où égale à la moyenne des valeurs du tableau passé en paramètre.

Exercice 4 :

Structure de choix :

Écrivez les fonctions suivantes :
1. isEmpty qui prend une chaine en paramètres est retourne s’il est vide ou non.
2. findLargest qui prend trois nombres en paramètres et renvoie le plus grand des trois.
3. isPrime qui prend un nombre en entrée et retourne vrai s'il s'agit d'un nombre premier, sinon faux.
